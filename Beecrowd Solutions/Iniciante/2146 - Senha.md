# 2146 - Senha

Escreva um programa que, dado um número, informe a respectiva senha.

[Senha - Beecrowd](https://www.beecrowd.com.br/judge/pt/problems/view/2146)

# Solução

```
#include <bits/stdc++.h>
using namespace std;

int main() {
    int n;
    while(cin >> n){
        n--;
        cout << n << "\n";
    }
}
```
