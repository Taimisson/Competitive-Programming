# 791A	Bear and Big Brother

https://codeforces.com/problemset/problem/791/A

# Solution

```
#include <iostream>
using namespace std;

int main() {
    int a, b;
    cin >> a >> b;

    int years = 0;
    while (a <= b) {
        a *= 3;
        b *= 2;
        years++;
    }

    cout << years << endl;
}
```
